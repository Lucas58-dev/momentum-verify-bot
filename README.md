# Momentum Verify Bot

Bot Discord pour vérifier les codes uniques et attribuer automatiquement le rôle Membre.

## Configuration

1. Crée un bot sur Discord et récupère le TOKEN
2. Crée une base Airtable avec les champs:
   - Email
   - Code Final (MOM-XXXXXX)
   - Utilisé (checkbox)
